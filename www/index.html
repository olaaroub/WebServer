Project Highlights

This repo implements the mandatory part (socket handling, parsing, request routing, CGI support basics). The cookies & sessions feature is planned next ‚Äî it's called out above so reviewers know it's on the roadmap.

HTTP parsing & routing

Robust parser for requests, support for routing and method handling (GET, POST).

CGI

Executes external programs to generate dynamic responses (configured via the assignment specs).

Static file serving

Efficient file read and correct Content-Length and MIME type responses.

Virtual hosts & config

Per-host configuration parsing to support several server blocks like the assignment.

Error handling

Custom 4xx/5xx pages and careful status code management for graders.

Extensible & tested

Designed with small, testable components ‚Äî easy to iterate and add sessions later.
