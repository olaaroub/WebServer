Project Highlights

This server implements all mandatory features, including socket handling, HTTP parsing, and CGI support, as well as the bonus features for cookies and session management.

HTTP Parsing & Routing

Robust parser for requests, with support for routing and method handling (GET, POST, DELETE).

CGI Execution

Executes external scripts (PHP, Python, Bash) to generate dynamic responses, with timeout handling.

Static File Serving

Efficiently serves static files with correct Content-Length and MIME type headers.

Virtual Hosts & Config

Per-host configuration parsing to support multiple server blocks, similar to Nginx.

Cookies & Sessions

Full support for session management via cookies, including creation, validation, and deletion.

Error Handling

Custom 4xx/5xx pages and careful status code management for all edge cases.
