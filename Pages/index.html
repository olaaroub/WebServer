Welcome to Webserv

A high-performance HTTP server written in C++98.

Server Features

Non-Blocking I/O

Built with epoll/kqueue for handling thousands of simultaneous connections efficiently.

HTTP/1.1 Compliant

Handles GET, POST, and DELETE methods, along with static file serving and CGI.

Customizable

Fully configured through a simple and powerful configuration file, just like Nginx.
